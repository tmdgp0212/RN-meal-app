# install
- `npm i -g expo-cli`

# create
- `npx create expo-app`

# commands
- `npm start`
  - a : `npm run android`
  - i : `npm run ios`
  - r : reload
  - j : open debugger
  - m : menu toggle
  - c : show QRcode
